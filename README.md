## Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas.
* Performed initial inspection (shape, columns, data types).

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions and relationships.
* Identified trends, patterns, and anomalies.
* Created visualizations for better understanding.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicates and inconsistencies.
* Standardized formats and corrected data types.

### 4. SQL Analysis (MySQL)

* Loaded cleaned data into MySQL server.
* Wrote SQL queries to extract insights.
* Performed aggregations, joins, and filtering.

### 5. Dashboard Creation

* Built an interactive dashboard in Power BI.
* Included key KPIs and visualizations.
* Designed for easy interpretation by stakeholders.

### 6. Reporting

* Summarized findings in a structured report.
* Highlighted key insights and recommendations.

### 7. Presentation (PPT)

* Created a professional presentation using Claude.
* Focused on storytelling and business impact.

## How to Run

1. **Clone the repository**

   ```bash
   git clone <your-repo-link>
   cd <project-folder>
   ```

2. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Run Python scripts**

   ```bash
   python main.py
   ```

4. **Set up MySQL**

   * Create a database
   * Import cleaned dataset
   * Run SQL queries from `/sql` folder

5. **Open Power BI Dashboard**

   * Load `.pbix` file in Power BI Desktop
